# coffee_shop_manager
Modular console application in Python to manage coffee orders and order history.

Aplicación de Cafetería en Consola

Aplicación de gestión de cafetería basada en consola desarrollada con Python.

!!Descripción¡¡

Esta aplicación permite a los usuarios:

Mostrar el menú de café

Realizar pedidos

Ver el historial de pedidos

Salir del sistema

El proyecto está estructurado utilizando múltiples módulos para mejorar la organización y escalabilidad.

!!Tecnologías¡¡

Python 3

!!Estructura del Proyecto¡¡

main.py → Punto de entrada de la aplicación

menu.py → Muestra las opciones del menú

pedido.py → Maneja la lógica de pedidos

historial.py → Gestiona el historial de pedidos

!!Cómo Ejecutarlo¡¡
git clone https://github.com/alexismartinon/coffee-shop-console-app.git
cd coffee-shop-console-app
python main.py
